# Shadowcore Studyink

A blazing-fast PDF viewer Chrome Extension with AI handwritten notes generation.

**Made by yeh.naitik**

## Features
- Canvas-based virtual PDF rendering (PDF.js) — no Google Viewer, no lag
- 1-finger draw, 2-finger scroll on touch devices
- Multiple brush types: pen, marker, pencil, calligraphy
- AI-powered handwritten notes generation
- Thumbnails, zoom, keyboard shortcuts, fullscreen
- Dark academic branding — Shadowcore Studyink

## Install
1. Download `shadowcore-studyink-v2.zip`
2. Unzip and go to `chrome://extensions`
3. Enable Developer Mode → Load Unpacked → select the `extension` folder
